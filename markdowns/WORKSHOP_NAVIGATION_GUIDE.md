# Workshop Navigation & UI Guide

## 🧭 Navigation Structure

### Accessing Workshop from Website

```
Homepage (/)
  └── Workshops Page (/workshops)
       └── Workshop Details (/workshops/[id])
            └── Join Live (/workshops/[id]/live)
```

### During Live Workshop

**Header Navigation Menu** (always visible):
- **Workshop Details** - Return to workshop info page
- **All Workshops** - Browse all available workshops  
- **Services** - View ministry services
- **Homepage** - Return to main website

**Access via**: Click "Navigate" button (←) in top-left corner

## 🎛️ Workshop Controls

### Pre-Join Screen (Waiting Room)

**Features**:
- Camera/microphone preview
- Audio level indicator
- Device selection dropdowns
- Test video/audio before joining
- Permission checks

**Actions**:
- **Join Workshop** - Enter live session
- **Retry** - Re-initialize if media fails
- **Cancel** - Return to workshop details

### Live Workshop Room

#### Main Controls (Bottom Bar)
- 🎤 **Mute/Unmute** - Toggle microphone
- 📹 **Video On/Off** - Toggle camera
- 🖥️ **Screen Share** - Share your screen (Presenter+)
- ✋ **Raise Hand** - Signal you want to speak
- 💬 **Chat** - Open chat panel
- 👥 **Participants** - View participant list
- 📊 **More** - Additional features
- 🚪 **Leave** - Exit workshop

#### View Layouts
- **Grid View** - Equal-sized tiles for all participants
- **Spotlight View** - Large main video + thumbnails
- **Sidebar View** - Video + side panels

#### Sidebar Panels
- **Chat** - Text messages
- **Participants** - Manage attendees
- **Polls** - Interactive voting
- **Q&A** - Questions and answers
- **Files** - Shared documents
- **More** - Additional features

#### Header Options (Top Right)
- 🔗 **Session Options** - Open in new tab, copy link
- ⚙️ **Settings** - Device settings, preferences
- ⛶ **Fullscreen** - Enter/exit fullscreen mode

## 👤 Role-Based UI Differences

### Guest (Limited Access)
- Can view video/audio only
- No camera/microphone sharing
- No chat or interactions
- Read-only experience

### Participant (Standard)
- Full video/audio participation
- Chat messaging
- Raise hand
- React with emojis
- View polls and Q&A

### Moderator
- All Participant features
- Mute/unmute others
- Remove participants
- Manage chat messages
- Create polls

### Presenter
- All Participant features
- Screen sharing
- Create polls
- Upload files
- Control whiteboard

### Co-Host
- All Presenter features
- Create breakout rooms
- Assign participants to rooms
- Record session
- Lock/unlock meeting
- Manage roles

### Host (Full Control)
- All Co-Host features
- End workshop for everyone
- Delete workshop
- Access all admin controls

- **View Results** - See real-time voting

## 🗂️ Breakout Rooms

### Creating Rooms
1. Click "Breakout Rooms" button
2. Set number of rooms
3. Choose assignment method:
   - Automatic (random)
   - Manual (drag & drop)
4. Set time limit (optional)
5. Click "Create Rooms"

### Managing Rooms
- **Join Room** - Enter specific breakout room
- **Broadcast** - Send message to all rooms
- **Close Rooms** - Return everyone to main room
- **Extend Time** - Add more minutes

## 📋 Participant Management

### View Participants
- See all active participants
- View participant roles
- See audio/video status
- View raised hands

### Manage Participants (Host/Co-Host)
- **Mute All** - Mute all participants
- **Mute Individual** - Mute specific person
- **Remove** - Kick participant from workshop
- **Change Role** - Promote/demote participant
- **Allow/Disallow Video** - Control video permissions

## 📹 Recording

### Start Recording (Co-Host/Host only)
1. Click "Record" button
2. Confirm recording notice
3. Recording indicator appears
4. All participants notified

### Stop Recording
1. Click "Stop Recording"
2. Recording saves automatically
3. Access via Admin Dashboard

## 🛠️ Troubleshooting

### Video Not Working
1. Check camera permissions in browser
2. Verify camera is not used by another app
3. Try different browser (Chrome recommended)
4. Refresh page and rejoin

### Audio Issues
1. Check microphone permissions
2. Test audio in waiting room
3. Switch microphone device
4. Check OS audio settings
5. Verify browser audio permissions

### Connection Problems
1. Check internet connection speed
2. Close other applications
3. Refresh browser
4. Try different network
5. Contact support if persistent

## 🔒 Privacy & Security

### Protected by Design
- End-to-end peer connections
- No data stored on central servers
- Recordings only if explicitly enabled
- Waiting room for host approval

### Best Practices
- Don't share workshop links publicly
- Use waiting room for added security
- Lock meeting when all participants joined
- Remove unwanted participants immediately

## 📱 Multi-Device Support

### Open in Multiple Tabs
- Click "Session Options" → "Open in New Tab"
- Same user can join from multiple devices
- Useful for dual monitor setups
- Share link between your devices

### Responsive Design
- Works on desktop, tablet, mobile
- Adaptive layouts for screen size
- Touch-optimized controls on mobile

## 🎯 Pro Tips

1. **Test first**: Join waiting room early to test equipment
2. **Good lighting**: Face a window or use desk lamp
3. **Stable internet**: Use wired connection when possible
4. **Headphones**: Reduce echo and background noise
5. **Mute when not speaking**: Keep audio clean
6. **Grid view for small groups**: See everyone equally
7. **Spotlight for presentations**: Focus on speaker
8. **Use reactions**: Engage without interrupting
9. **Pin video**: Keep important participant always visible
10. **Keyboard shortcuts**: Learn them for efficiency
